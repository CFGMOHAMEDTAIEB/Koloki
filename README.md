
# Coloki - Secure Colocation Platform

A comprehensive platform connecting renters and house owners with enterprise-level security, transparent pricing, and built-in monetization.

## Features

- **Secure User Verification**: KYC verification, background checks
- **Protected Transactions**: Escrow-based payment system with 10% platform commission
- **Advertising Platform**: Earn revenue through publisher partnerships
- **Listing Fees**: 10 DT listing fee for house owners
- **Policies & Legal**: Comprehensive T&C, privacy policy, and dispute resolution
- **Mobile & Web**: Responsive design for all devices
- **Admin Dashboard**: Moderation and analytics tools

## Getting Started

### Prerequisites
- Node.js 18+ 
- npm or yarn

### Installation

```bash
npm install
```

### Development

```bash
npm run dev
```

The app will be available at `http://localhost:5173`

### Build for Production

```bash
npm run build
npm run preview
```

## Security

This platform is built with security as a top priority:
- End-to-end encryption for sensitive data
- Secure authentication (JWT + OAuth)
- GDPR/CCPA compliance
- PCI DSS compliance for payments
- Regular security audits
